Practice project utilizing React for frontend view
 - vite as build tool
 - CSS modules
 - Material UI
 - React Router
 - Redux for state management
 - React hook form for forms

Also Java as backend
- tomcat server
- servlets
- jdbc, jdbi for database access
- postgres as relational database
